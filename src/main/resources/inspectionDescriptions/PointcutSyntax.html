Reports invalid pointcut expression syntax in AOP advice and @Pointcut annotations.

Checks for:

  • Empty pointcut expressions
  • Expressions that contain only negation operators
  • Unbalanced parentheses
  • Leading or trailing logical operators (&&, ||)
  • Consecutive logical operators (for example && ||)
  • Missing logical operator between pointcut clauses
  • Unknown pointcut designators (e.g., typos like exectuion instead of execution)

Valid designators: execution, within, this, target, args, @annotation, @within, @target, @args, bean, and other supported AspectJ designators.
